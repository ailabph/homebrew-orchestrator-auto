# Ailabph Orchestrator-auto

## How do I install these formulae?

`brew install ailabph/orchestrator-auto/<formula>`

Or `brew tap ailabph/orchestrator-auto` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ailabph/orchestrator-auto"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
